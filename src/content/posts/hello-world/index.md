# 你好，世界！

欢迎来到我的博客！这是一个使用 [Astro](https://astro.build) 构建的静态博客站点。

## 为什么选择 Astro？

Astro 是一个专注于内容的网站框架，它有以下优势：

1. **零 JavaScript 默认** - 只在需要时发送 JavaScript
2. **内容集合** - 方便管理 Markdown/MDX 内容
3. **组件化** - 支持多种 UI 框架
4. **性能优异** - 首屏加载速度极快

## 技术栈

- **框架**: Astro
- **样式**: 原生 CSS + CSS Variables
- **部署**: Vercel / Netlify
